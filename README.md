# udemy-project

This project is a responsive Udemy-inspired course platform UI built using HTML and CSS. The main objective of this project was to practice modern layout techniques using Flexbox, including properties such as flex-grow, flex-basis, and alignment.

The website includes sections like Recommended Courses, Recommended Topics, and Popular Courses, where each course card displays course details, ratings, and pricing. The project also includes a structured footer section for navigation and additional information.

Through this project, I improved my skills in layout design, responsive structure, UI arrangement, and Flexbox positioning.

Technologies Used:

HTML5

CSS3

Flexbox (flex-grow, flex-basis, alignment)

Key Features:

Course listing with ratings and price

Recommended courses section

Recommended topics section

Popular courses display

Footer section for additional links
